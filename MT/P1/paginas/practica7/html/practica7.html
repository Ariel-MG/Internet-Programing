> Práctica 7 - Aplicaciones Web con PHP

Aplicación Dockerizada con PHP y MySQL

Esta práctica implementa una aplicación web utilizando PHP y MySQL ejecutándose en contenedores Docker. La aplicación incluye dos calculadoras matemáticas que demuestran el uso de algoritmos para aproximar constantes matemáticas importantes.

Tecnologías Utilizadas

  • Docker & Docker Compose: Containerización de la aplicación
  • PHP: Lenguaje de programación del backend
  • MySQL: Base de datos relacional
  • Apache: Servidor web
  • HTML/CSS: Frontend y estilos

Arquitectura del Proyecto

  • docker-compose.yml: Configuración de servicios
  • html/: Código fuente PHP
  • mysql_data/: Persistencia de datos

Calculadoras Disponibles

π

Aproximación de Pi (π)

Utiliza la serie de Leibniz para calcular aproximaciones del número π (pi) mediante iteraciones. Cuantas más iteraciones, mayor precisión en el cálculo.

Fórmula: π = 4 × Σ((-1)ⁿ/(2n+1))
Calcular π
e

Aproximación de e (Euler)

Calcula aproximaciones del número e (base de los logaritmos naturales) usando la serie de Taylor. Mayor número de iteraciones proporciona mayor precisión.

Fórmula: e = Σ(1/n!)
Calcular e

Instrucciones de Uso

1

Iniciar los Servicios

Ejecuta docker-compose up -d en el directorio de la práctica para levantar los contenedores.

2

Acceder a las Aplicaciones

Haz clic en los botones de las calculadoras para acceder a las aplicaciones PHP en el navegador.

3

Realizar Cálculos

Introduce el número de iteraciones deseado y observa cómo se aproxima a los valores reales de π y e.
